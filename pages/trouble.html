접속 불가, 다운로드 실패, 속도 저하 등 자주 발생하는 오류와 해결법을 정리했습니다.

자주 발생하는 오류 해결법

증상을 클릭하면 해결 방법을 확인할 수 있습니다.

① DNS 변경
인터넷 서비스 제공사(ISP) 차단이 원인일 수 있습니다. DNS를 구글(8.8.8.8) 또는 클라우드플레어(1.1.1.1)로 변경 후 재시도하세요.

② 브라우저 캐시 삭제
Ctrl+Shift+Delete → 캐시·쿠키 삭제 후 재접속하세요.

③ 다른 브라우저 시도
크롬, 엣지, 파이어폭스 등 다른 브라우저로 접속을 시도해보세요.

④ 서버 점검 여부 확인
서비스 공식 SNS나 공지사항에서 점검 일정을 확인하세요.

① 접속기 사용 여부 확인
무료 접속기는 저속입니다. 유료 고속 접속기를 사용하면 속도가 크게 향상됩니다.

② 피크 시간대 피하기
저녁 9~11시는 트래픽이 집중되는 시간입니다. 오전이나 낮 시간대에 다운로드하면 더 빠릅니다.

③ 동시 다운로드 수 줄이기
여러 파일을 동시에 받고 있다면 하나씩 순서대로 다운로드하세요.

④ 인터넷 연결 상태 확인
WiFi보다 유선 연결이 안정적입니다. 공유기 재시작도 효과적입니다.

① 접속기 재시작
접속기 프로그램을 완전히 종료 후 재시작하고 다시 다운로드를 시작하세요.

② 임시 파일 삭제
이전에 실패한 다운로드의 임시 파일(.tmp 등)이 남아있으면 충돌할 수 있습니다. 임시 파일을 삭제 후 재시도하세요.

③ 백신·방화벽 예외 등록
보안 프로그램이 다운로드를 차단하는 경우가 있습니다. 웹하드 접속기를 백신 예외 목록에 추가하세요.

① 관리자 권한으로 실행
설치 파일을 우클릭 → '관리자 권한으로 실행'으로 다시 시도하세요.

② 기존 버전 완전 삭제 후 재설치
제어판 → 프로그램 제거에서 기존 접속기를 완전히 삭제한 후 최신 버전을 새로 설치하세요.

③ Windows Defender 예외 추가
Windows 보안 설정에서 접속기 설치 경로를 예외로 추가하면 설치가 원활해집니다.

① 로그아웃 후 재로그인
충전 직후에는 잔액이 바로 반영되지 않을 수 있습니다. 로그아웃 후 다시 로그인하면 정상 반영됩니다.

② 고객센터 문의
재로그인 후에도 코인이 보이지 않는다면 충전 영수증(결제 내역)을 첨부해 고객센터에 문의하세요.

① 파일 확장자 확인
.mkv, .avi 등 파일 형식에 맞는 플레이어가 필요합니다. 곰플레이어, 팟플레이어, VLC 등을 설치하세요.

② 파일 손상 여부 확인
다운로드가 완료되지 않은 상태에서 파일을 열면 오류가 발생합니다. 다운로드가 100% 완료됐는지 확인하세요.

③ 압축 파일 처리
분할 압축 파일(.zip, .rar, .001 등)은 모든 파트를 같은 폴더에 모아두고 첫 번째 파일을 압축 해제하세요.

💬 해결이 안 된다면 고객센터 문의

위 방법으로도 해결되지 않는다면 이용 중인 웹하드 서비스의 고객센터에 문의하세요. 넷파일은 1:1 문의 기능을 통해 빠른 지원을 받을 수 있습니다.
